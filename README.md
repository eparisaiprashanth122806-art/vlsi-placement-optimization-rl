# AI-Based VLSI Placement Optimization using Reinforcement Learning

This project demonstrates how Reinforcement Learning (DQN) can be used to optimize VLSI circuit placement by minimizing wirelength.

## Features
- Reinforcement Learning (DQN)
- Wirelength optimization
- 3D Visualization
- Animation of optimization
- ISCAS benchmark circuits

## How it works
- Circuit → Nodes (gates) + Edges (connections)
- Agent moves components
- Reward = -wirelength - overlap penalty
